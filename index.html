Session 2 • NotebookLM · inside the tool

Create your first notebook

  1. Go to notebooklm.google.com and sign in.
  2. Click Create new.
  3. Add sources — drag-and-drop, a URL, or Google Drive.
  4. Wait for NotebookLM to index your sources.
  5. Start chatting, or use one-click tools in the Studio panel.
notebooklm.google.com → your notebook
The real thing — sources added, and you just ask: “Can you summarize…”
Tips · name it subject + unit (e.g. DBMS – Unit 3) · one notebook = one topic → sharper, better-cited answers.

Session 2 • Hands-on · Task 2

Explore the document — generate these ten

  1. A 100–150 word summary of the document.
  2. The five most important concepts discussed.
  3. Three difficult terms explained in simple language.
  4. Five MCQs with answers.
  5. Five flashcards (question → answer) for revision.
  1. One-page revision notes for the whole document.
  2. Five interview / viva questions (with model answers).
  3. The key learning outcomes.
  4. A mind map or concept hierarchy (describe it if visuals aren't available).
  5. A 7-day study plan to master the document.
Prompt pattern: “From my sources only, …” + the task + a word / format limit.
